## PID 이란?

피드백 제어 방식 중 가장 일반적인 비례제어 방식

비례, 적분, 미분을 사용하여 목표값에 가깝게 하는 제어이다.

- P 동작 : Proportional 동작, 오차에 대해 비례적으로 제어
- I 동작 : Intergral 동작, 오차의 적분에 대해 제어
- D 동작 : Differential 동작, 오차의 미분에 대해 제어(바로 전 편차를 비교)

### 언제 사용할까?

균형을 유지해야하는 상황

- 드론이 바람 등으로 인해 피격 받았을 때 균형 유지

### 방식?

> P 제어
> 

목표값과 현재 값의 차이에 비례하여 조작랼을 조절한다.

오차의 정도가 input에 1차 함수로 적용된다. (y = ax)

![image.png](attachment:d305e95b-ee66-4bf8-9bf4-ce3e29db2043:image.png)

- P 제어식 : input = error * pgain
- 여기서 사용하는 상수를 p gain이라 하며, 이 상수의 값에 따라 오차에 대한 input 정도가 달라진다.

![이 그래프는 응답 곡선으로, 시간에 대한 output의 자취 곡선을 뜻한다.](attachment:72ffba77-c8be-4f50-95a5-f5f0891ecd46:image.png)

이 그래프는 응답 곡선으로, 시간에 대한 output의 자취 곡선을 뜻한다.

- 하지만 p제어는 현재 output이 목표 output에 도달하지 못하고 진동하는 오실레이션 현상이 발생한다.

![image.png](attachment:adbe263e-a5d4-4562-b33a-c8b497e00337:image.png)

P제어의 오실레이션 단점을 보완했지만 목표치에 맞추기 위해 편차를 축적하는 시간이 좀 걸린다.

> PD 제어
> 

PI와 같이 p제어의 오실레이션 단점을 보완한 형태

- input = error * P gian + error(d) * D gain
    - 이때 error(d)는 오차의 미분값으로 이전 대비 오차의 증감이다.

- 조작량에 민첩하고 빠르게 조작량에 반응한다.
- 측정값 = 설정값 이 되게 하는 방식

### 제어게인과 프로세스 게인

- 제어게인 : 제어를 할 수 있는 능력
- 프로세스 게인 : 성능의 정도 (이동 능력?)

프로세스 게인을 참고하여 제어게인을 판단한다.

예시 ) A차량의 최고 속도가 100km이고, B차량의 최고속도가 200km 일때 제어 게인 범위가 60km~80km 이면, 상대적으로 A보다 B가 제어하는 범위가 적으니까 B가 제어게인이 낮다고 할 수 있음
