## UDP란?

데이터 전송시에 신뢰성은 없지만 빠른 속도를 제공하는 프로토콜

실시간에 유용하다!

LAN 환경에서 실시간 처리 용으로 사용된다.

보내는 패킷의 단위는 데이터그램이다.

빅엔디안을 사용하지만 x86/ARM CPU는 리틀엔디안을 사용하기에 변환이 필요하다.

### 패킷 구조

헤더가 8바이트

| 보내는쪽 포트 | 2B |
| --- | --- |
| 받는 쪽 포트 | 2B |
| 길이 | 2B |
| 체크섬 | 2B |

메모리를 적게 먹어서 가볍다.

### 왜 신뢰성이 없는데도 사용할까?

공부하다 든 생각이었다. 그래서 찾아보니

유튜브를 볼 때 영상이 멈추는 것보단 화질이 깨진채로 이어나가는게 훨씬 나은것처럼

실시간으로 상태를 확인하는게 중요한 환경에서 잘 사용한다.

반대인 TCP로 하면 상태확인이 느리기 때문에 답답해질수있기 때문이다.
