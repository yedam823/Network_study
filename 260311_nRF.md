## RF 모듈?

일단 RF는? Radio Frequency

무선 통신 모듈의 종류 중 하나로서, 간단한 데이터 송수신부터 무선으로 장치를 컨트롤할 수 있게 해주는 모듈

2.4GHz 대역을 사용하고, 통신가능 거리가 길다!

(안테나를 달면 더 길어진다.)

근데 벽이나 물체같은 장애물이 있을 때, 수신율이 떨어짐

### 원리

송신하고자 하는 정보를 전파로 변조한 후 전송을 하면 수신하는 모듈에서 전파를 또 복조하여 정보를 나타냄

### 특징

- SPI 통신을 사용한다.
- 다대다 통신, 1 대 다 통신 둘 다 가능하다.

### 아두이노로 실습했을 때의 코드

```c
#include <SPI.h>
#include <nRF24L01.h>
#include <RF24.h>
RF24 radio(8, 10); // CE, CSN
const byte address[6] = "00001";
void setup() {
  radio.begin();
  radio.openWritingPipe(address);
  radio.setPALevel(RF24_PA_MIN);
  radio.stopListening();
}
void loop() {
  const char text[] = "Hello World";
  radio.write(&text, sizeof(text));
  delay(1000);
}
```
