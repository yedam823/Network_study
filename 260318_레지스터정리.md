## 레지스터 정리

| 카테고리 | 레지스터 이름 | 기능 |
| --- | --- | --- |
| RCC | RCC_APBxENR, RCC_AHBxENR | GPIO나 UART를 사용하기 전에 클럭 리셋, 활성화를 함 |
| GPIO | GPIOx_MODER | 핀 모드 설정 |
|  | GPIOx_ODR | 출력 데이터 (HIGH, LOW 출력) |
|  | GPIOx_BSRR | 비트 셋/리셋 (ODR 보완) |
|  | GPIOx_IDR | 입력 데이터(HIGH,LOW 읽기) |
| SysTick | STK_LOAD | 정확한 시간을 만들기 위해 리로드값 설정 |
|  | STK_CTRL | SysTick 제어 (타이머 인터럽트 발생시킬지 안시킬지) |
| UART | USARTx_SR / ISR | 상태 나타냄, 통신 병목을 줄이기 위해 플래그를 인터럽트나 DMA로 처리 |
|  | USARTx_DR / TDR, RDR | 데이터 전송, 수신 레지스터 |

## RMW?

Read Modify Write

읽기, 수정, 쓰기

값을 읽어오고, 연산을 하거나, 원래 위치에 저장함

주의할 점 : 여러 프로세스나 스레드가 동시에 같은 데이터에 RMW 하려고 하면 경쟁상태(Race Condition)가 발생함 (병목)

→ 이를 막기 위해 원자적(Atomic) 연산이 중요해짐
