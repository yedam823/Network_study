## 모듈형 확장

시스템의 하드웨어나 소프트웨어를 처음부터 다시 설계하지 않고도 필요한 기능을 선택저긍로 추가하거나 교체할 수 있도록 만든 구조

→ 유지보수가 편하고 유연성이 좋아서 중요하다.

### 소프트웨어적 관점

- LKM (Loadable Kernel Module) - 시스템이 돌아가고 있는 중에 insmod 명령어로 확장해서 붙일 수 있음
    - 리눅스 커널이 대표적
    - 커널 크기 경량화가 가능하고 OS 재설치 안해도됨

### 하드웨어적 관점

- 보드 설계 시 하나의 기판에 다 넣지 않고 분리하는 방식
    - SoM(System on Module) - CPU, 메모리 같은 핵심 부품은 작은 모들에 담고, 인터페이스는 베이스보드(Carrier Board)에 두어 CPU 모듈만 업그레이드 할 수 있게 함
    - 우리가 자주 쓰는 드라이버 모듈도 이거지…

→ 모듈을 나눠서 잘 사용해보자
