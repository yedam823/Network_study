## 드론 시점 객체 탐지 시스템

드론 시점에서 보이는 카메라 영상 속 객체를 YOLOv8을 통해 분석하고 무엇인지 탐지해 그 위치를 추적하는 시스템이다.

### 목적

AI가 매우 중요하다는 걸 뼈저리게 깨달은 후… AI를 도입한 드론 시뮬레이터를 만들고 싶었고, 이중에서도 영상 분석 기술을 배워두면 좋을 것 같다는 생각으로 제작했다.

### 어떤 객체를 탐지하나?

탱크를 탐지한다. 발전시켜나가며 자주포도 탐지할 것이다.

### 어떻게 분석할까?

1. 합성 데이터를 활용한다.
    1. 데이터 수집은 Roboflow Universe에서 military vehicle aerial 공개 데이터셋을 가져올 것이다.
    2. ArmA 3에서 훈련 데이터 생성을 추후에 해볼 것이다.

2. 드론이 너무 높은 고도에 있을 때, 이미지를 조각내어 추론하는 SAHI 기법을 적용한다. 

### 추가할 기술

1. 트래킹하는 기술, 사라져도 예상경로를 예측한다.
    1. ultralytics 내장기능인 ByteTrack를 사용
    2. 경로 예측은 filterpy 사용
2. 위장, 은폐된 타겟에 대한 내성을 키운다.
    1. Albumentations?
3. 엔진 열기를 감지하는 걸 키운다. (열화상 감지)
    1. FLIR ADAS Dataset? 추후에 필요에 따라 추가할 것이다.
