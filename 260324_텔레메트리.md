## 텔레메트리

원격 측정 기술

서버, 애플리케이션, 모니터링 장치와 같은 원격 소스에서 데이터를 수집하고 전송 및 분석을 수행하는 프로세스이다.

1. 데이터 수집
2. 데이터 전송
3. 데이터 분석 및 모니터링

→ 근데 이를 달성하려면 일단 데이터가 있어야한다.

이 데이터를 어떻게 가져올까?

### 데이터 유형

- 로그 : 로그 기록한 게 텔레메트리 데이터가 될수있다.!
- 추적 경로 : 요청이 시스템을 통해 이동할 때 남기는 이동경로를 수집한다. 근본 원인 분석이나 병목현상 해결에 도움을 줌
- 메트릭 : 런타임에 측정된 시스템의 수치 측정량이다. 성능 분석 가능

### 단점

- 로그, 경로, 메트릭 이런것이 밖에 노출된다면 해킹 위험이 증가한다. → 암호화, 액세스 제어 등이 필요하다.

### 사용 분야

- 항공 / 드론
- 위성 / 우주
