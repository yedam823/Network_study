## Hardfault

예외 처리 메커니즘, 처리가 불가한 경우 발생되는 메세지이다.

이것이 발생하면 복구가 불가능한 시스템 오류라고 생각하면 쉽다.

### 원인?

1. 복구가 불가능한 시스템
2. 원인 파악이 불가능한 메모리
3. 버스 폴트, 페이지 폴트 등 핸들러가 실행이 불가하다고 판단되는 경우
4. 빈 페이지가 없을 경우
5. 정해진 수치보다 적을 경우

시스템에 따라 종류는 다양하다.

### Hardfault ≠ memory management fault, usage fault … ?

- 후자의 fault의 경우, 지역이나 노조의 시위…
- 전자의 fault의 경우, 대통령 파면…

→ 후자의 fault가 발생했는데, 이를 처리할 handler가 활성화 되지 않았을 때 하드폴트가 발생한다.

### 조치 방법

디버깅을 통해서 열심히 파악해야한다 (어렵다)

디버깅 프로그램 (ST-Link, Jtag) 을 잘 공부해보자
